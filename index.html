Objetivos

Al finalizar el curso se espera que las y los estudiantes puedan:
* Crear paginas web usando HTML, CSS, SVG y Javascript.
* Escribir el 100% del codigo en un sitio Web
* Diseñar una web que se adapte a diferentes tamaños de pantallas y gestionar la navegacion.
* Crear elementos animados que interactuen con el usuario
* Consumir API´s con Javascript para colocar contenidos en un sitio

Contenido

UNIDAD 1: HTML y Desarrollo Web

Entorno de desarrollo. Editor de codigo. Inspector y herramientas de desarrollo de los navegadores. Funcionamiento basico de un sitio web. elementos de un sitio web. Etiquetas y atributos. Etiquetas de texto. Imagenes y enlaces. Etiquetas de bloque. Etiquetas de documentos. Repositorios y Git.

UNIDAD 2: Introduccion a las CSS

Vincular un archivo CSS. Sintaxis basica de CSS. Selectores. Estilos y topografias. Estrucutras en las hojas de estilo. Selectores avanzados. Estilos de navegacion. Restes CSS. El modelo de la caja. Displays. Floats. Positions. Estilos para listas. Border, border-radius y box-shadow. Fondos e imagenes de fondo. WebFronts. Colores y opacidad. Bordes y redondeados. Sombras. Degradados y fondos.
